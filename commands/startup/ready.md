Run the Fundraise Readiness assessment from `~/.claude/skills/startup-metrics/SKILL.md` (Stage 8: Fundraise Readiness).

1. Determine what stage they'd be raising at (pre-seed, seed, series A, B)
2. Walk through the readiness scorecard — score each dimension with evidence
3. Compare their metrics to VC expectations for that stage
4. Run the red flags checklist — flag any fundraise killers
5. Give a total score with clear verdict (ready / almost / not yet / too early)
6. If not ready, identify the 2-3 things to fix first and estimate time to fix

Don't tell every founder to raise. Some should bootstrap. If unit economics are strong and growth is steady, raising might not be the right move.
