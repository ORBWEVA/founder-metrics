Run the Funnel Math analysis from `~/.claude/skills/startup-metrics/SKILL.md` (Stage 5: Funnel).

1. Ask what their funnel stages are (or use the default: visitor → signup → activated → paying → retained)
2. Get numbers for each stage
3. Calculate conversion rates between each stage
4. Compare to benchmarks for their model (freemium, free trial, sales-led)
5. Identify the biggest drop-off (largest gap vs benchmark)
6. Run impact modeling: show the revenue impact of improving each stage by a realistic amount
7. Recommend which stage to fix first (activation > retention > conversion > traffic)

The insight is always: "fixing THIS stage has more impact than anything else." Make that unmistakably clear.
