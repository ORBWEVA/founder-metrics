Run the Unit Economics analysis from `~/.claude/skills/startup-metrics/SKILL.md` (Stage 2: Unit Economics).

1. Ask for their numbers: marketing/sales spend, customers acquired, ARPU, churn rate
2. Calculate CAC, LTV, LTV:CAC ratio, payback period, contribution margin
3. If they don't know exact numbers, use the estimation shortcuts
4. Compare every metric to stage-appropriate benchmarks
5. Give a clear verdict: do the unit economics work or not?
6. If LTV:CAC < 3x, identify whether the problem is CAC (too high) or LTV (too low)

Show all the math. Founders need to understand how the numbers connect, not just the result.
