---

Run the PMF assessment from `~/.claude/skills/startup-metrics/SKILL.md` (Stage 1: Product-Market Fit).

1. Ask what product they're assessing
2. Ask if they have Sean Ellis survey data — if yes, calculate the score
3. If no survey data, ask for retention data (weekly or monthly cohorts)
4. If no retention data, walk through the Pre-PMF leading indicators checklist
5. Run segment analysis — break down by user type, plan, industry, or activation status
6. Identify which segment (if any) has PMF even if the overall score doesn't
7. Give a clear verdict and specific next action

Don't sugarcoat. "Almost there" is not PMF. Either the number is above 40% or it isn't.
