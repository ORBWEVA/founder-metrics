Show the user the available startup metrics commands:

```
Startup Metrics Commands
=========================
/startup:pmf        — Product-Market Fit assessment (Sean Ellis + retention)
/startup:unit       — Unit economics (CAC, LTV, payback, margins)
/startup:churn      — Churn diagnosis (logo vs revenue, cohorts, recovery)
/startup:runway     — Burn rate + runway projection with scenarios
/startup:funnel     — Funnel math + bottleneck identification
/startup:growth     — Growth curve analysis (J-curve, hockey stick, etc.)
/startup:moat       — Competitive moat assessment (7 moat types)
/startup:ready      — Fundraise readiness scorecard
/startup:glossary   — Explain any startup term with your real numbers
/startup:help       — This help message

Recommended flow:
  Validate:  pmf → unit → funnel
  Sustain:   churn → growth → moat
  Fundraise: ready → (fix gaps) → ready again

Install: npx skills add ORBWEVA/founder-metrics
Repo:    github.com/ORBWEVA/founder-metrics
```
